Network-as-a-Service Architecture with SDN and NFV

A Proposed Evolutionary Approach for Service Provider Networks

M.P.V. Manthena
TU Delft - Network Architectures and Services
Mentors: F.A. Kuipers, C. Van den Broek
SDN, NFV, Network Virtualization, NaaS

Research Overview

Comprehensive thesis addressing Internet ossification through evolutionary NaaS implementation using SDN and NFV technologies. Includes proof-of-concept implementation and experimental performance evaluation on physical network testbeds.

Key Contributions:
  • Evolutionary approach to NaaS implementation
  • SDN and NFV integration strategies
  • Proof-of-concept physical testbed
  • Performance evaluation and validation
